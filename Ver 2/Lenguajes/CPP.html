C++

La programacion es el proceso de diseñar, codificar, depurar y mantener el codigo de los programas computacionales, con el proposito de crear programas que ejecuten acciones especificas deseadas. Se divide en los siguientes pasos:

  1. Entender el Problema
  2. Plantear y Elegir Soluciones
  3. Construir Algoritmo
  4. Codificar y Probar
  5. Mantener y Mejorar

Se debe entender y resolver el problema antes de codificarlo. El problema se puede a su vez estructurar en: entradas de variables, procesamiento de operaciones y salidas de resultados.

Una vez resuelto el problema se establece una lista ordenada de acciones necesarias para llegar a dicha solucion, esto se denomina algoritmo. Los lenguajes de programacion son los lenguajes artificiales que nos permiten comunicar nuestros algoritmos a las computadoras.

Cada lenguaje cuenta con sus propios simbolos y reglas que conforman su estructura y significado. Los lenguajes de programacion se dividen en interpretados y compilados. Los lenguajes interpretados pueden ser ejecutados de inmediato, mientras que los compilados necesitan pasar por una revision sintactica y semantica del codigo fuente antes de generar el codigo ejecutable.

Todo programa en C se puede dividir en una estructura fundamental compuesta por una seccion de librerias, una seccion de variables y una seccion de instrucciones. Las librerias agregan nuevas instrucciones, las variables son simbolos con nombres unicos que representan valores y las instrucciones son las tareas ejecutadas por el programa.

Los nombres de las variables deben comenzar con una letra y no contener espacios o caracteres especiales, a excepcion del guion bajo. Todas las instrucciones se deben incluir dentro de la funcion especial main().

Printf() es una instruccion que nos permite mostrar texto (cadena de caracteres) o el valor de las variables en la consola de comandos. Para introducir espacios con este comando utilizamos el simbolo de escape (n). Para mostrar el valor de variables utilizamos la representacion que corresponda a su tipo de dato y luego el nombre o valor de la variable separada por una coma.

printf("String %d", variableInt);

Scanf() es la instruccion que le permite al usuario definir el valor de una variable en el tiempo de ejecion.

scanf("%d", &variableInt);

Normalmente se ejecutan las instrucciones en orden, de la primera a la ultima. Este proceso se denomina Ejecucion en Cascada. Para modificar el flujo de ejecucion de las instrucciones se utilizan Estatutos de Control. Los estatutos de control de decicion son If y Else. Los estatutos de control de repeticion o ciclos son While, Do y For. Los estatutos de control necesitan que se cumpla una condicion para tomar la decicion de ejecutar una seccion de codigo o de repetirla.

Los Conectores Logicos nos permiten establecer mas de una condicion para que los estatutos de control tomen su decision.
