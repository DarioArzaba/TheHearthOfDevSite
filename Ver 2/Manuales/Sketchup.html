P = Push Pull para extruir Los elementos se pegan automaticamente a los puntos cuando utilizas la herramienta de mover. Shift Key al mover bloquea la direccion a la que se ha pegado el objeto Ctrl Key es para copiar el objeto Utilizar (X) para activar XRay Outliner nos permite seleccionar y organizar objetos con Xray activado Los objetos en 3d se componen de figuras geometricas basicas las cuales podemos agrupar a nuestra conveniencia. Todos los grupos que son copiados de un componente son modificados automaticamente, esencial para grupos que se repiten. El Fallow me tool nos permite crear un Path que seguran los objetos para una extruccion Los puntos basicos de geometria son esquinas, lineas y puntos medios. Shift con select tool nos permite seleccionar varias entidades La herramienta de ofset crea una copia geometrica de la cara que esta seleccionada, si utilia alguna de las lineas seleccionadas entonces solo utiliza las lineas. Selecciona Zoom y luego Cambia el Field of View para ver mejor dentro de objetos. DoubleClick con Offset recuerda la ultima medida que utilizamos para introducir la geometria. Paint Tool Alt para Sample de color. Shift para colorear todas las caras pegadas del mismo color. ------------------------ Sketchup es poligonal en vez de curbed based NURBS. Utiliza caras (faces) como superficies y tiene poca capcidad para trabajar con solidos. Todos los modelos estan conformados por lineas y caras. El ambiente se divide en: Ventana de Modelado: Siepre da una representacion 3D del modelo actual. La barra de menus: Nos da acceso a todos los comandos del programa Toolbars: Al igual que la barra de menus nos da acceso a los comandos pero de una forma mas rapida con iconos grandes. La Barra de estado: Da informacion pertinente al comando que se esta ejecutando. Paneles y Tray: Es el panel a la dereche que presenta algunos apartados con informaicon util.
